## 🚀 Быстрый старт

### 1. Настройка переменных окружения

```bash
cd E:\github\ArFintes2\backend_tg
copy .env.example .env
```

Отредактируйте `.env` и установите:
- `DATABASE_URL` - строка подключения к PostgreSQL
- `TELEGRAM_BOT_TOKEN` - токен вашего бота от @BotFather
- `JWT_SECRET` - любой секретный ключ (минимум 32 символа)

### 2. Установка зависимостей

```bash
npm install
```

### 3. Настройка базы данных

**Вариант A: Использование Prisma**
```bash
npx prisma generate
npx prisma migrate dev --name init
npx prisma db seed
```

**Вариант B: Автоматическое создание (при запуске)**
Просто запустите сервер в режиме разработки - таблицы создадутся автоматически.

### 4. Запуск сервера

```bash
# Режим разработки (с авто-перезагрузкой)
npm run dev

# Сборка для продакшена
npm run build
npm start
```

## 🌐 API Endpoints

### Авторизация
- `POST /api/auth/login` - Вход/регистрация через Telegram
- `POST /api/auth/verify` - Проверка токена

### Пользователь
- `GET /api/user/profile` - Получить профиль
- `PUT /api/user/profile` - Обновить профиль
- `GET /api/user/stats` - Статистика
- `POST /api/user/daily` - Ежедневная награда
- `GET /api/user/referrals` - Рефералы

### Кейсы
- `GET /api/cases` - Список кейсов
- `POST /api/cases/open` - Открыть кейс
- `GET /api/cases/history` - История открытий

### Инвентарь
- `GET /api/inventory` - Список предметов
- `POST /api/inventory/combine` - Сборка скина из фрагментов
- `POST /api/inventory/sell` - Выставить на продажу

### Рынок
- `GET /api/market` - Список лотов
- `POST /api/market/listings` - Создать лот
- `POST /api/market/buy` - Купить лот
- `POST /api/market/listings/:id/cancel` - Отменить лот
- `GET /api/market/history` - История продаж

### Каналы
- `GET /api/channels` - Список каналов
- `POST /api/channels/check-subscriptions` - Проверить подписки
- `POST /api/channels/claim-reward` - Получить награду

### Реальные скины
- `GET /api/real-skins` - Список скинов
- `POST /api/real-skins/withdraw` - Заявка на вывод

### Мини-игры
- `GET /api/games/games` - Список игр
- `POST /api/games/dice/play` - Кости
- `POST /api/games/roulette/play` - Рулетка
- `POST /api/games/slots/play` - Слоты
- `POST /api/games/coinflip/play` - Орёл и решка

### Платежи
- `GET /api/payments/packages` - Пакеты премиума
- `POST /api/payments/create` - Создать платёж
- `GET /api/payments/status/:id` - Статус платежа
- `GET /api/payments/history` - История платежей

### Админка
- `GET /api/admin/stats` - Статистика платформы
- `GET /api/admin/users` - Пользователи
- `POST /api/admin/users/:id/balance` - Изменить баланс
- `GET /api/admin/withdrawals` - Заявки на вывод
- `POST /api/admin/withdrawals/:id/status` - Статус заявки

## 🐛 Известные ограничения

1. **Платежные системы** - только демо режим. Для продакшена нужна интеграция с реальными API.

2. **Проверка подписок Telegram** - используется случайная генерация. Для продакшена нужна реальная проверка через Telegram Bot API.

3. **Rate limiting** - упрощенная реализация. Рекомендуется использовать Redis для продакшена.
