# ⚖️ Настройки Экономики (Баланс)

## 💰 Валюты

### CR (Credits) - Обычная валюта
```
Ежедневный вход:
- День 1: 100 CR
- День 2: 100 CR
- День 3: 100 CR + бонус 50 CR
- День 7: 100 CR + бонус 200 CR (Streak!)
- Максимум в день: 500 CR (с бонусами)

Продажа скина:
- Обычный: 100-500 CR
- Редкий: 500-2000 CR
- Легендарный: 2000-10000 CR

Награды в мини-играх:
- Победа в дуэли: ставка x1.9
- Рулетка (цвет): ставка x2
- Рулетка (число): ставка x36
- Слоты (2 одинаковых): ставка x2
- Слоты (3 одинаковых): ставка x4-10
- Орёл/Решка: ставка x1.95
```

### GC (Gem Coins) - Премиум валюта
```
Покупка за реальные деньги:
- 100 GC = 99₽ (бонус 10%)
- 500 GC = 450₽ (бонус 20%)
- 1000 GC = 890₽ (бонус 30%)
- 5000 GC = 3990₽ (бонус 50%)

Награды:
- Реферал (прямой): 200 GC
- Реферал (2 уровень): 50 GC
- Реферал (3 уровень): 20 GC
- Подписка на канал: 50-200 GC
- Топ недели (1 место): 1000 GC
```

## 👥 Реферальная Система

### Уровни:
```
1 уровень (прямые рефералы):
- Бонус за регистрацию: 200 GC
- Комиссия с покупок: 20%
- Комиссия с проигрышей: 5%

2 уровень:
- Комиссия с покупок: 10%
- Комиссия с проигрышей: 2%

3 уровень:
- Комиссия с покупок: 5%
- Комиссия с проигрышей: 1%
```

### Пример заработка:
```
Вы пригласили 10 человек:
- Бонус за регистрацию: 10 × 200 GC = 2000 GC

Рефералы купили GC на 10,000₽:
- Ваша комиссия: 20% = 2000 GC

Рефералы проиграли в играх 50,000 CR:
- Ваша комиссия: 5% = 2500 CR

Итого: 2000 GC + 2500 CR
```

### Лимиты:
```
Максимум рефералов 1 уровня: 1000
Минимальная активность реферала: 1 игра или 1 кейс за 7 дней
Иначе реферал не засчитывается
```

## 📈 Прогноз Экономики

### При 1000 пользователей:
```
Ежедневная активность:
- 60% заходят ежедневно (600 человек)
- 40% открывают кейсы (240 человек)
- 20% играют в мини-игры (120 человек)
- 5% покупают GC (50 человек)

Доход в день:
- Покупки GC: 50 × 300₽ = 15,000₽
- Комиссии: 5,000₽
- Спонсоры: 2,000₽
Итого: 22,000₽/день = 660,000₽/месяц

Расход в день:
- Сервер: 70₽
- Награды (GC эквивалент): 10,000₽
- Поддержка: 5,000₽
Итого: 15,070₽/день = 452,100₽/месяц

Прибыль: 207,900₽/месяц
```
